muspy.representations¶

Representations¶

This module provides functions for converting a Music object to and from common representations.

muspy.representations.to_event_representation(music)[source]¶

Convert a muspy.Music object to an event-based representation.

Parameters

music (muspy.Music object) – Music object to convert.

muspy.representations.to_note_representation(music)[source]¶

Convert a muspy.Music object to a note-based representation.

Parameters

music (muspy.Music object) – Music object to convert.

muspy.representations.to_pianoroll_representation(music)[source]¶

Convert a muspy.Music object to a pianoroll representation.

Parameters

music (muspy.Music object) – Music object to convert.

muspy.representations.to_representation(music, target)[source]¶

Convert to a target representation.

Parameters

  • music (muspy.Music object) – MusPy Music object to be converted. The file format is inferred from the extension.

  • target (str) – Target representation. Supported values are ‘event’, ‘note’, ‘pianoroll’ and “piano-roll”.
