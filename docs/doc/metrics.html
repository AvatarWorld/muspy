muspy.metrics¶

Metrics¶

This module provides functions for computing common metrics on a Music object. This can be useful for analyzing datasets and evaluating generative models.

muspy.metrics.empty_beat_rate(music)[source]¶

Return the empty beat rate.

The empty beat rate is defined as the ratio of the number of empty beats (where no pitch is played) to the number of beats.

\[empty\_beat\_rate = \frac{\# of empty beats}{\# of beats}\]
Parameters

music (muspy.Music object) – Music object to be evaluate.

Returns

Empty beat rate.

Return type

float

muspy.metrics.in_scale_rate(music, root, mode)[source]¶

Return the rate of notes in a musical scale.

In scale rate is defined as the ratio of the number of notes in a scale to the total number of time steps.

Parameters

  • music (muspy.Music object) – Music object to be evaluate.

  • root (int) – Root of the scale.

  • mode (int) – Mode of the scale.

Returns

In scale rate.

Return type

float

muspy.metrics.n_pitch_classes_used(music)[source]¶

Return the number of unique pitch classes used.

Parameters

music (muspy.Music object) – Music object to be evaluate.

Returns

  • int – Number of unique pitch classes used.

  • list of bool – A boolean list of length 12 indicating the pitch classes used. The first element corresponds to pitch A, and the last one corresponds to pitch G# (G sharp).

muspy.metrics.n_pitches_used(music)[source]¶

Return the number of unique pitches used.

Parameters

music (muspy.Music object) – Music object to be evaluate.

Returns

  • int – Number of unique pitch classes used.

  • list of bool – A boolean list of length 128 indicating the pitches used. If the i-th element is True, the MIDI note pitch i is used.

Return the pitch range.

muspy.metrics.polyphony_rate(music, threshold=2)[source]¶

Return the polyphony measure.

The polyphony is defined as the ratio of the number of time steps where multiple pitches are on to the total number of time steps.

\[polyphony = \frac{ \# of time steps where multiple pitches are on}{\# of time steps}\]
Parameters

music (muspy.Music object) – Music object to be evaluate.

Returns

Polyphony rate.

Return type

float
