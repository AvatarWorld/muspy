class muspy.classes.Base(**kwargs)[source]¶

Base class for MusPy objects.

It implements the following three handy methods.

  • Intuitive and meaningful __repr__ in the form of class_name(attr_1=value_1, attr_2=value_2,…).

  • Method from_dict that sets the attributes by the corresponding values in a dictionary.

  • Method to_ordered_dict that returns the object as an OrderedDict.

Notes

This is the base class for MusPy objects. To add a new class, please inherit from this class and set the class variables _attributes properly. The list _attributes contains all the attribute keys of the class so that the above methods can be implemented.
