muspy.core¶

Core functions¶

Core functions that can be applied to a muspy.Music object.

muspy.core.adjust_resolution(music, target=None, factor=None)[source]¶

Adjust resolution and update the timing of time-stamped objects.

Parameters

  • music (muspy.Music object) – MusPy music object to be adjusted.

  • target (int, optional) – Target resolution.

  • factor (int or float, optional) – Factor used to adjust the resolution based on the formula: new_resolution = old_resolution * factor. For example, a factor of 2 double the resolution, and a factor of 0.5 halve the resolution.

muspy.core.adjust_time(obj, func)[source]¶

Adjust the timing of time-stamped objects.

Parameters

  • obj (muspy.Music or muspy.Track object) – Object to be adjusted.

  • func (callable) – The function used to compute the new timing from the old timing, i.e., new_time = func(old_time).

See also

muspy.core.append(obj1, obj2)[source]¶

Append an object to the correseponding list.

  • If obj1 is of type muspy.Music, obj2 can be muspy.KeySignature, muspy.TimeSignature, muspy.Lyric, muspy.Annotation or muspy.Track.

  • If obj1 is of type muspy.Track, obj2 can be muspy.Note, muspy.Lyric or muspy.Annotation.

  • If obj1 is of type muspy.Timing, obj2 can be muspy.Tempo.

Parameters

  • obj1 (muspy.Music, muspy.Track or) – muspy.Tempo object Object to which obj2 to be append.

  • obj2 (MusPy objects (see below)) – Object to be appended to obj1.

muspy.core.clip(obj, lower=0, upper=127)[source]¶

Clip the velocity of each note.

Parameters

  • obj (muspy.Music, muspy.Track or muspy.Note) – object Object to be clipped.

  • lower (int or float, optional) – Lower bound. Defaults to 0.

  • upper (int or float, optional) – Upper bound. Defaults to 127.

muspy.core.quantize(music, resolution, beats=None, bpm=None, offset=0)[source]¶

Quantize the timing of time-stamped objects.

Parameters

  • music (muspy.Music object) – MusPy music object to be quantized.

  • resolution (int) – Time steps per beat.

  • beats (list of float) – Sorted list of beat positions in ascending order, assuming no duplicate values.

  • bpm (int or float) – Length of quantization step, in seconds.

  • offset (float) – Offset of the beat pulse train, e.g., start time of the first beat.

muspy.core.sort(obj)[source]¶

Sort all the time-stamped objects with respect to event time.

  • If a muspy.Music is given, this will sort key signatures, time signatures, lyrics and annotations, along with notes, lyrics and annotations for each track.

  • If a muspy.Track is given, this will sort notes, lyrics and annotations.

  • If a muspy.Timing is given, this will sort tempos.

Parameters

obj (muspy.Music, muspy.Track or) – muspy.Timing object Object to be sorted.

Transpose all the notes by a number of semitones.

Parameters

  • object – Object to be transposed.
