muspy.outputs¶

Output Interfaces¶

MusPy provides the following output interfaces.

  • Save losslessly to JSON or YAML: muspy.save_json() and muspy.save_yaml()

  • Write to other formats: muspy.write_midi() and muspy.write_musicxml()

  • Convert to other objects: muspy.to_pretty_midi() and muspy.to_pypianoroll()

  • Wrappers: muspy.save(), muspy.write() and muspy.to_object()

muspy.outputs.save(music, path)[source]¶

Save a Music object loselessly to a JSON or a YAML file.

Parameters

  • music (muspy.Music object) – MusPy Music object to be saved.

  • path (str or Path) – Path to save the file. The file format is inferred from the extension.

See also

muspy.write()

Write to other formats such as MIDI and MusicXML.

Notes

The conversion can be lossy if any nonserializable object is used (for example, in an Annotation object, which can store data of any type).
